# 러스트란?

누구나 탄탄하고 효율적인 소프트웨어를 만들게 해주는 프로그래밍 언어

### 1. 고성능

- 대단히 빠르고 메모리효율적
- 별도의 런타임이 없음
- GC 없음

### 2. 신뢰성

- 강력한 타입시스템
- 소유권 모델로 메모리 + 스레드 안전
- 컴파일 시점에 문제 예방 가능

### 3. 생산성

- 훌륭한 문서
- 친절한 컴파일러
- 훌륭한 패키지 매니저와 빌드툴
- IDE 자동완성, 타입검사, 자동포맷 등 지원
